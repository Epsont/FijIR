# FijiR Road Map

- [ ] **Расширить меню с кнопками**
  - Добавить раскрывающийся список для перехода в Кадастровую карту с использованием разных ресурсов.
- [ ] **Реализовать чтение зума**
  - Использовать считанный зум для перехода в Яндекс и кадастровые карты.
- [ ] **Реализовать сохранение выбранной кнопки**
  - Сохранять выбор пользователя из раскрывающихся списков.
- [ ] **Реализовать чтение домена**
  - Генерировать кнопку для перехода на кадастровую карту в зависимости от текущего домена.
